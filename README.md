# SparkVCFtools

``Hadoop 3.1.1 & Spark 2.4.5``

* VCF Merge

* gVCFCombine

### **For use**

* gVCF

``python3 gvcfCombine_info.py --appname [test] --c [20] <<- info file!``

``python3 gvcfCombine_sample.py --infoName [test] --c [20] <<- sample file!``

* VCF 

``python3 vcfmerge.py --appname [test]``
